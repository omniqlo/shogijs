A JavaScript Shogi library

## Installation

```sh
$ npm install @omniqlo/shogijs
# or
$ yarn add @omniqlo/shogijs
```

## Setup

```sh
$ git clone git@github.com:omniqlo/shogijs.git
$ cd shogijs
$ npm run setup
```
